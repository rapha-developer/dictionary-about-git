git remote add origin {.git}

Conecta o repositório remoto, acessado pelo {.git} ao repositório local

git branch -M {name}

Substitua a branch atual {ex: master} com um novo nome definido pelo usuário {ex: main}

git push -u origin main

Empurra as atualizações da branch local {ex: main} para o repositório remoto {ex: origin}
